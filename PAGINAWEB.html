La estructura de toda página web

Toda página HTML tiene siempre la misma base. Es como el esqueleto del cuerpo.

mipagina.html
<HTML>               ← empieza todo
  <HEAD>             ← cabecera (no se ve en pantalla)
    <TITLE>Mi página</TITLE>  ← título de la pestaña
  </HEAD>
  <BODY>             ← aquí va todo lo visible
    ... todo el contenido ...
  </BODY>
</HTML>              ← termina todo
💡 Qué hace cada parte
<HTML>
Le dice al navegador que esto es una página web
<HEAD>
Información sobre la página. No se muestra en pantalla.
<TITLE>
El texto que aparece en la pestaña del navegador.
<BODY>
Todo lo que se ve en pantalla va aquí dentro.

Las etiquetas básicas

Una etiqueta es una instrucción entre corchetes angulares. Casi todas tienen apertura y cierre.

<H1>...<H6>

Encabezados

Títulos de diferente tamaño. H1 es el más grande, H6 el más pequeño.

<P>

Párrafo

Agrupa un bloque de texto con espacio arriba y abajo.

<B> y <I>

Negrita y cursiva

Negrita para resaltar. Cursiva para énfasis.

<HR>

Línea horizontal

Dibuja una línea de separación. Sin cierre.

<BR>

Salto de línea

Baja a la siguiente línea sin nuevo párrafo.

<CENTER>

Centrar

Centra el contenido en la pantalla.

<FONT>

Fuente y color

Cambia tamaño (SIZE) y color (COLOR) del texto.

<IMG>

Imagen

Inserta una foto con SRC="nombre.jpg".

BGCOLOR

Fondo de página

Atributo de BODY para cambiar el color de fondo.

Ejemplo
<H1><CENTER>Mi primera página</CENTER></H1>
<HR>
<P>Texto <B>negrita</B> y <I>cursiva</I></P>
<FONT COLOR="RED" SIZE=5>Texto rojo grande</FONT>

Tablas

Organizan información en filas y columnas. TABLE → TR → TH o TD.

Estructura
<TABLE BORDER=1 CELLPADDING=6>
  <TR>  <TH>Nombre</TH>  <TH>Nota</TH>  </TR>
  <TR>  <TD>Ana</TD>    <TD>9</TD>    </TR>
</TABLE>
TR=fila  TH=cabecera  TD=celda  COLSPAN=unir columnas

Formularios

  <INPUT TYPE="CHECKBOX">  ← varias
  <TEXTAREA></TEXTAREA>  ← area grande
  <INPUT TYPE="SUBMIT">    ← boton enviar

GitHub y GitHub Pages

GitHub guarda y comparte código. GitHub Pages publica tu página web gratis con URL pública.

¿Qué es GitHub?

Un "Google Drive para código"

Guarda proyectos, muestra el historial de cambios y permite trabajar en equipo. Lo usan millones de programadores.

GitHub Pages

Hosting gratuito

Convierte tu repositorio en una web real con URL pública. Gratis para siempre, sin publicidad.

Cómo publicar tu página en 5 pasos:

1

Crea cuenta en github.com

Haz clic en "Sign up" y regístrate gratis con tu correo.

2

Crea un repositorio nuevo

Clic en + → "New repository". Ponle nombre, márcalo como Public y activa "Add a README file".

3

Sube tu archivo HTML

Add file → Upload files. Súbelo renombrado como index.html y haz clic en "Commit changes".

4

Activa GitHub Pages

Settings → Pages. En Branch selecciona main y / (root). Guarda.

5

Tu URL está lista

En 2 minutos aparece: https://tunombre.github.io/mi-repositorio. Compártela con quien quieras.

💡
¿Por qué es importante?
Antes necesitabas pagar un servidor para tener una web. Con GitHub Pages cualquiera puede publicar gratis en minutos. Es la herramienta que usan desarrolladores de todo el mundo.
